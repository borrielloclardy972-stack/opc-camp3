# OPC超级个体训练营 GitHub Pages 版本

当前版本为 UI V3：登录页重构、首页、代理认证卡密、培训营物料、学习进度、考核学习进度。

## 默认演示卡密

```text
用户登录：OPC-VIEW-2026
管理员登录：OPC-EDIT-2026
```

## 页面模块

1. 首页
2. 代理认证卡密
3. 培训营 / 训练营
4. 学习进度
5. 考核学习进度
6. 注册申请 / 上传素材 / 删除素材

GitHub Pages 是静态托管，注册、上传、删除、统计数据当前为前端演示。后续需要真实跨设备同步时，再接 Cloudflare Worker 或其他后端。
