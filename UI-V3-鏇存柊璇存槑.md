# OPC超级个体训练营 UI V3 更新说明

## 本次已调整

### 1. 登录页

- 重新设计登录页，不再使用单一居中小卡片。
- 改为“左侧品牌/学习收益展示 + 右侧登录注册卡片”的 SaaS 官网式布局。
- 保留三个入口：用户登录、管理员登录、注册申请。
- 登录页文案突出：引流训练、销售/交付训练、代理认证、学习闭环。

### 2. 侧边栏顺序

已改为：

1. 首页
2. 代理认证卡密
3. 培训营 / 训练营
4. 学习进度
5. 考核学习进度
6. 切换卡密

### 3. 首页

新增首页模块，先作为待定内容位，包含：

- 公司/项目介绍占位
- 学员能学到什么
- OPC使命：共创、共担、共利、共生
- 学员成长路径：注册、学习、实操、开单、考核

### 4. 代理认证卡密

包含：

- 账号认证状态
- OPC超级个体身份展示
- 等级制度：青铜、白银、黄金、铂金、钻石、王者
- 分数度
- 开单数量
- 开单金额
- 综合排名
- 代理排名看板

### 5. 培训营 / 训练营

二级菜单保留并可筛选右侧素材：

- 引流端培训营
- 交付端 / 销售端培训营
- 产品核心物料
- 基础文档 / SOP
- 高转化素材案例
- 成交 / 交付案例
- 工具清单 / FAQ

### 6. 学习进度

独立成单独模块，根据课程维度展示：

- 引流端培训营学习进度
- 销售端培训营学习进度
- 交付端培训营学习进度
- 产品物料资料学习进度

### 7. 考核学习进度

已独立出规划中模块，保留入口和业务说明：

- 完成引流端培训
- 完成销售课培训
- 完成交付端培训
- 进入综合考核
- 验证是否掌握引流、销售、交付与核销核心技能

```text

```
