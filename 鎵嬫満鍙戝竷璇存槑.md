# OPC超级个体训练营发布说明

## 你后面发给代理什么最方便

最方便的形式是一个 H5 链接，例如：

`https://你的域名/opc`

代理用手机点开链接，输入卡密即可进入：

- 查看卡密：`OPC-VIEW-2026`
- 编辑卡密：`OPC-EDIT-2026`

编辑卡密会显示「上传素材」和「批量导入」入口。

## 当前版本能做到什么

- 手机浏览器可打开
- 可输入查看/编辑两种卡密
- 编辑模式可从手机选择图片、视频、PDF、Office 文件
- 图片素材上传后会成为素材卡片主图
- 可添加到手机桌面，像一个轻量 App
- 未配置云端时，上传内容保存在当前手机/浏览器本地
- 配置 Cloudflare Worker 后，上传内容会同步到云端，所有代理打开同一个链接都能看到

## 发布成手机链接的最省事方式

推荐使用 Cloudflare Pages 的 Direct Upload：

1. 登录 Cloudflare
2. 进入 Workers & Pages
3. Create application
4. 选择 Pages
5. 选择 Drag and drop
6. 上传整个 `模版库` 文件夹
7. 发布后会得到一个 `pages.dev` 链接
8. 把链接发给代理

## 云端共享版怎么开

项目里已经包含云端后端文件：

- `cloudflare-worker.js`
- `wrangler.toml.example`
- `cloud-config.js`

你需要做一次 Cloudflare 配置：

1. 创建一个 KV Namespace，绑定名叫 `MATERIALS_KV`
2. 创建一个 R2 Bucket，绑定名叫 `MATERIALS_BUCKET`
3. 部署 `cloudflare-worker.js`
4. 设置两个 Worker Secret：
   - `VIEW_CODE`
   - `EDIT_CODE`
5. 部署成功后复制 Worker 地址
6. 打开 `cloud-config.js`
7. 把 `apiBase` 改成 Worker 地址，例如：

```js
window.OPC_CLOUD = {
  apiBase: "https://opc-api.your-name.workers.dev"
};
```

这样就是正式共享版：

- 查看卡密：只能看
- 编辑卡密：能上传
- 编辑人员上传后，所有代理都能看到

## 重要区别

### 轻量版

只发布当前前端文件夹，不配置 `apiBase`。

优点：最快，手机能点开，能输入卡密，能本机上传。

限制：每个代理上传的素材只保存在自己的手机浏览器里，别人看不到。

### 正式共享版

配置 `cloud-config.js` 里的 `apiBase`。

优点：一个人上传，所有人都能看到；卡密由云端校验；素材文件可存到 R2。

## 当前文件说明

- `index.html`：页面结构
- `styles.css`：视觉样式
- `app.js`：筛选、卡密、上传、本地保存逻辑
- `cloud-config.js`：云端接口配置
- `cloudflare-worker.js`：云端共享后端
- `wrangler.toml.example`：Cloudflare Worker 部署配置示例
- `manifest.webmanifest`：手机桌面应用配置
- `sw.js`：离线缓存
- `assets/icons/`：平台 App 图标和 OPC 图标
